PROJETOS

" MyBigWhale " - Idealizador e Coordenador

Sendo um conjunto de ferramentas online de edição de arquivos, imagens e documentos, o site foi fundado em 2023 com o objetivo de ser uma ferramenta simples e prática para todos os tipos de usuários.

O projeto foi construído em arquitetura monolítica, assim o projeto poderia se manter de maneira sustentável em meio aos desafios de se criar um produto opensource.

" Significando " - Criador

O projeto é uma solução web para auxiliar os usuários a mapearem os significados que eles atribuem às mais diversas palavras, sentimentos e emoções da lingua portuguesa.

Além da iniciativa pautada em problemáticas que o uso intenso da internet causa nas pessoas, o projeto também busca criar um sistema de visualização dos dados de forma lúdica, criando uma experiência única aos usuários.
